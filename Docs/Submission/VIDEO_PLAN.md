# LOADED Gameplay Video Plan

## Video Requirements

- Length: 30초에서 60초
- 실제 플레이 화면만 사용
- AI 합성 플레이 영상 사용 금지
- 공개 또는 일부 공개 YouTube 업로드

## Target Length

45초

## Shot List

| 시간 | 장면 | 전달할 내용 | 준비 상태 |
|---|---|---|---|
| 0초에서 5초 | 적 접근과 무작위 장전 | 핵심 규칙 | Pending |
| 5초에서 12초 | 이동과 방향 변경 | 위치 판단 | Pending |
| 12초에서 20초 | 탄창 전체 발사 | 순서 기반 콤보 | Pending |
| 20초에서 30초 | 탄환 보상과 덱 수정 | 로그라이크 성장 | Pending |
| 30초에서 42초 | 보스전 | 발사 타이밍과 위험 관리 | Pending |
| 42초에서 45초 | 타이틀과 링크 | 프로젝트 인지 | Pending |

## Recording Checklist

- [ ] 프레임 드랍 없음
- [ ] 디버그 UI 제거
- [ ] 실제 플레이만 사용
- [ ] 조작이 화면에서 명확히 보임
- [ ] 게임 핵심이 10초 안에 나타남
- [ ] 외부 음악 권리 확인

## 작성 예시

촬영 후 준비 상태를 `Recorded`, 편집 완료 후 `Ready`로 바꾸고 파일명과 테이크 번호를 장면 설명에 추가한다.
