# NAN 2026 Submission Checklist

완료 표시는 실제 제출 파일과 공개 상태를 별도 환경에서 검증한 뒤 변경한다.

## 1. 플레이 가능한 빌드 및 소스 코드

- [ ] Web 빌드 링크에서 즉시 실행 가능
- [ ] GitHub Pages 또는 정적 호스팅 배포
- [ ] PC Chrome 테스트
- [ ] PC Edge 테스트
- [ ] Android Chrome 테스트
- [ ] 가능한 경우 iOS Safari 테스트
- [ ] 별도 유료 라이선스 불필요
- [ ] 게임 전체 소스 코드 포함
- [ ] `Assets`, `Packages`, `ProjectSettings` 포함
- [ ] `.meta` 파일 포함
- [ ] 커밋 기록 유지
- [ ] 저장소 공개 여부 확인
- [ ] 비공개인 경우 심사 계정 초대
- [ ] 심사 계정: dl_gameai_reviewer@nhn.com

## 2. 플레이 동영상

- [ ] 30초에서 60초
- [ ] 실제 게임 플레이 화면
- [ ] AI 조작 또는 합성 영상 미사용
- [ ] 타인 영상 미사용
- [ ] YouTube 업로드
- [ ] 공개 또는 일부 공개 설정
- [ ] 영상 URL 확인

## 3. 게임 소개 및 설명 PDF

- [ ] 게임 제목
- [ ] 한 줄 소개
- [ ] 게임 목표
- [ ] 조작 방법
- [ ] 종료 조건
- [ ] 실행 방법
- [ ] 플레이 링크
- [ ] 플레이 영상 링크
- [ ] PDF 열람 테스트

## 4. AI 활용 기술 PDF

- [ ] AI 도구 목록
- [ ] 모델 또는 버전
- [ ] AI 활용 구조
- [ ] 주요 프롬프트
- [ ] 주요 지시 사항
- [ ] AI 출력 요약
- [ ] 검증 방식
- [ ] 직접 수정한 영역
- [ ] 실패 및 폐기 사례
- [ ] 외부 에셋 출처
- [ ] 오픈소스 출처
- [ ] PDF 열람 테스트

## 5. 최종 QA

- [ ] 링크 권한 확인
- [ ] 시크릿 브라우저에서 실행
- [ ] 새 기기에서 실행
- [ ] 저장 데이터 초기화 테스트
- [ ] 게임 시작부터 종료까지 플레이
- [ ] 모든 URL 클릭 테스트
- [ ] 제출 파일 이름 확인
