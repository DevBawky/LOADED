# LOADED AI 활용 기술 문서

## 작성 지침

NAN 2026 AI 활용 기술 PDF의 원고다. 각 사례를 [AI Usage Log](../AI/AI_USAGE_LOG.md)의 실제 ID와 연결하고, 성공뿐 아니라 실패와 직접 수정 사항도 증거와 함께 기록한다.

## 1. 프로젝트 개요

LOADED는 무작위 탄환 순서와 전체 발사 타이밍을 다루는 턴제 탄창 빌딩 로그라이크다.

## 2. AI 활용 목표

설계 검토, 구현 보조, 테스트 도출, 밸런스 분석, 문서화를 가속하되 최종 판단과 검증은 개발자가 수행한다.

## 3. 사용한 AI 도구

| 도구 | 모델 | 사용 목적 | 주요 결과 |
|---|---|---|---|
| Codex | 확인 후 기록 | 문서 체계 초기화 | 설계, 개발, AI, 제출 기록 템플릿 |

## 5. 전체 AI 활용 구조

문제 정의 -> 프롬프트 작성 -> AI 결과 생성 -> 결과 검토 -> 테스트 및 검증 -> 직접 수정 -> 실제 프로젝트 반영
