# LOADED Balance Log

## Balance Principles

- 완벽한 탄창보다 불완전한 탄창의 판단을 강조한다.
- 무작위성은 결과를 결정하는 것이 아니라 판단 상황을 만든다.
- 기본탄도 특정 상황에서는 의미가 있어야 한다.
- 특정 탄환이 없으면 진행 불가능한 전투를 만들지 않는다.
- 평균 플레이 시간은 10분에서 15분을 목표로 한다.

## Metrics

- 평균 전투 턴
- 평균 장전 횟수
- 평균 발사 횟수
- 탄창이 가득 찬 상태에서의 발사 비율
- 빈 탄창 발사 시도 비율
- 탄환별 덱 포함률
- 탄환별 승률
- 전투별 피해량
- 무효 탄환 비율
- 최대 연쇄 처치
- 평균 런 시간
- 보스 도달률
- 클리어율
