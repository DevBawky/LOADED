# LOADED AI Asset Log

생성형 AI로 만든 그래픽, 사운드, 영상 후보를 생성 즉시 기록한다. 최종 미사용 결과도 폐기 사유를 남긴다.

| ID | 날짜 | 도구 및 모델 | 에셋 | 프롬프트 또는 참조 | 사용 위치 | 수정 내용 | 상태 |
|---|---|---|---|---|---|---|---|
| AIA-000 | YYYY-MM-DD | 도구와 모델 | 작성 예시 | 프롬프트 ID 또는 요약 | TBD | 리터치 내용 | Candidate |
