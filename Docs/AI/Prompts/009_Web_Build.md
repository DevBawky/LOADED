## Initial Prompt

```text
Unity 기반 LOADED 프로젝트를 Web 플랫폼으로 배포하기 위한 준비 사항을 검토한다.

목표:
- GitHub Pages 또는 정적 호스팅에서 링크 클릭 후 바로 플레이
- PC Chrome과 Edge, Android Chrome 지원
- 가능한 경우 iOS Safari 지원
- 별도 유료 라이선스 없이 실행
- 모바일에서도 버튼과 텍스트가 잘 보일 것

Unity Web 빌드 설정, 압축, 캐시, 메모리, 화면 비율, 터치 입력, 포커스 손실, 오디오 자동 재생 제한, 로컬 저장, 초기 로딩, 빌드 크기, GitHub Pages 구조, MIME 타입, 미지원 API, 오류 로그 수집을 검토하라.

코드를 바로 수정하지 말고 먼저 체크리스트와 예상 위험을 작성하라. 현재 프로젝트 파일을 확인한 뒤 실제로 필요한 항목만 제안하고, 확인하지 못한 값은 추측하지 마라.
```
