## Initial Prompt

```text
LOADED의 적 행동 시스템을 설계한다. 기본 전장은 일자형이며 적은 플레이어 행동 이후 한 번 행동한다.

요구사항:
- 적의 다음 행동을 플레이어에게 미리 표시할 수 있어야 한다.
- 이동, 근접 공격, 원거리 공격, 방어, 대기를 지원한다.
- 모든 적이 복잡한 AI를 사용하지 않아도 된다.
- 동일한 상태에서는 동일한 행동을 선택하도록 결정론적으로 재현한다.
- 플레이어가 이해하기 어려운 숨겨진 규칙을 줄인다.
- 행동 로직과 시각 연출을 분리한다.
- 일반 적 5종 이내로도 전투 다양성을 만들 수 있어야 한다.

먼저 최소 적 유형과 각 적이 만드는 전술적 질문을 제안하라. 이후 데이터 구조와 행동 결정 흐름을 작성하라.
```
