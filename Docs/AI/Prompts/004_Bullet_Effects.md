- 새 탄환 추가 시 기존 코드 수정을 최소화한다.
- Web 빌드에서 사용할 수 없는 기능에 의존하지 않는다.

먼저 데이터 구조, 실행 컨텍스트, 결과 데이터, 효과 인터페이스를 제안하라.
