## Initial Prompt

```text
LOADED의 핵심 게임 구조를 검토한다.

게임은 일자형 턴제 전투이며 플레이어 행동은 이동, 방향 변경, 무작위 탄환 장전, 장전된 탄환 전체 발사의 4개뿐이다. 장전은 보유 탄환 덱에서 무작위로 이루어지고, 탄환 순서는 바꿀 수 없다. 발사하면 모든 탄환이 앞에서부터 순서대로 발동한다. 전투 후에는 탄환 획득, 제거, 복제, 강화를 통해 덱을 구성한다.

다음 목표를 기준으로 핵심 루프의 강점, 위험, 불필요한 복잡성을 검토하라.

- 플레이 영상 10초 안에 규칙 전달
- 직관적인 PC 및 모바일 조작
- 10분에서 15분 세션
- 최소한의 그래픽 리소스로 높은 전투 피드백
- 반복 플레이와 빌드 다양성

핵심 행동을 추가하지 말고, 결정되지 않은 규칙은 질문 또는 TBD로 남겨라. 제안은 문제, 이유, 최소 변경안, 검증 방법 순서로 작성하라.
```
