# LOADED Prompt Records

## Recording Rules

1. 파일명은 `NNN_Topic_Name.md` 형식을 사용한다.
2. 초기 프롬프트, 추가 지시, 결과 요약, 검증, 직접 수정 사항을 분리한다.
3. 비밀 정보와 개인정보를 기록하지 않는다.
4. 관련 [AI Usage Log](../AI_USAGE_LOG.md) ID와 실제 파일을 연결한다.

## Prompt Record Template

```md
# P000 Prompt Title

## Metadata

- Date:
- Author:
- Tool and Model:
- Related AI Usage ID:
- Related Files:

## Initial Prompt

프롬프트 원문 또는 핵심 지시를 기록한다.

## Follow-up Instructions

- 추가 지시와 변경 이유

## Output Summary

## Validation

## Human Modifications

## Decision
```
